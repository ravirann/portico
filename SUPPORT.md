# Support

## Questions

Use GitHub Discussions for questions, ideas, and implementation guidance.

## Bugs

Open a GitHub issue with:

- Portico version or commit SHA.
- Environment details.
- Expected behavior.
- Actual behavior.
- Reproduction steps.
- Logs, traces, and screenshots with secrets removed.

## Security issues

Do not open public issues for vulnerabilities. See [SECURITY.md](SECURITY.md).
